Cada módulo está diseñado para una función específica del flujo de datos operativos de Claro & VTR.

📡
01 ↗
Paautin — Claro
Carga masiva desde la API CGSA con INSERT IGNORE por caseNumber. 87 campos por interacción.
API_Paautin 87 campos INSERT IGNORE
🔧
02 ↗
Sabana TOA — VTR
Descarga automatizada desde Oracle Field Service vía Selenium. Login SSO Microsoft + notificación WhatsApp.
Selenium Oracle Cloud WhatsApp API
📂
03 ↗
Carga Manual
Drag & drop de CSV y Excel hasta 150 MB. Normalización automática de columnas. Append o replace.
CSV · XLSX 150 MB Auto-normalize
⚠️
04 ↗
Mesa Error 40
Gestión de errores ONF desde Excel local o Google Sheets público. Upsert inteligente por rango de fechas.
Google Sheets Upsert MesaError40
📊
05 ↗
Análisis COD
Visualización de CodigoCierre con gráficos HTML5 Canvas. Filtros por red, exportación PNG y Excel.
Canvas Charts Filtros red Export Excel
🔍
06 ↗
Consulta de Tablas
Explorador visual con selector de campos, preview de 50 filas y exportación Excel de hasta 100k registros.
100k filas Preview 3 tablas
📑
07 ↗
Reporte PowerPoint
Genera automáticamente una presentación PPTX de 7 slides cruzando Paautin + Error40 + TOA por mes.
7 slides python-pptx Auto-download
📋
08 ↗
Auditoría IA
4 casos/semana por ejecutivo. Agente IA (Gemini/Claude) evalúa 5 criterios ponderados automáticamente.
Gemini 2.0 Claude Sonnet 5 criterios
⚙️
09 ↗
Scheduler
APScheduler para ejecución automática de Paautin y TOA. Zona horaria America/Santiago. Config JSON.
APScheduler Cron jobs America/Santiago
